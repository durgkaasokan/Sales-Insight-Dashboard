# Sales Insight Dashboard

## Overview
This project focuses on analysing sales data to uncover business insights, sales trends, and performance patterns through data cleaning, transformation, and analytical reporting techniques using Excel.

The project includes:
- Raw sales dataset
- Cleaned and processed dataset
- Final analytical report summarising findings and outcomes

---

## Project Files

| File | Description |
|---|---|
| `Sales_Insights_Dashboard.xlsx` | Sales dataset containing analysis and dashboard work |
| `Sales_Insights_Summary.pdf` | Final report summarising key findings and insights |
